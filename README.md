
# Household Energy Usage Forecast Using Machine Learning.

PowerPulse is a machine learning project designed to forecast household energy consumption using historical data. The project explores time-series modeling, feature engineering, and regression algorithms to build predictive models.


##  Project Objectives
- Preprocess and clean household energy usage data

- Engineer meaningful features like time-based trends and rolling averages

- Train and evaluate multiple regression models

- Visualize energy consumption patterns and model performance
## Technologies Used

- Python: Data processing and modeling

- Pandas & NumPy: Data manipulation

- Matplotlib & Seaborn: Data visualization

- Scikit-learn: Machine learning (Linear Regression, Random Forest, etc.)

- Jupyter Notebook: Interactive development
## Key Features

- Handles missing values and outliers

- Adds rolling average and time-based features

- Supports multiple regression algorithms

- Visualizes actual vs. predicted energy usage
## 📁 Dataset

The dataset includes time-stamped power consumption readings with features like:

- Global_active_power

- Global_reactive_power

- Voltage
